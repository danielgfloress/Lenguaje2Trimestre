## 🖥️ Descripción del proyecto

Nuestra página web sigue los partidos de fútbol de los equipos del momento (Raimon, Royal Academy, 
Academia Zeus, Wild, Farm, ...). Mostrando sus estadisticas, su posicionamiento en el campo a la 
hora de jugar, entre muchos datos de los equipos e individualmente.
También existe la tienda oficial de recuerdos y productos de los equipos y partidos que han jugado 
junto con entradas de próximos partidos.

## ⚙️ Estructura y descripción de uso

* **CSS** - Un CSS común para todo el proyecto, usado por todos los integrantes del grupo,
            donde hemos combinado estilos para desarrollar una página en común, con su correcto
            declaramiento de etiquetas.
* **HTML** - Cada miembro del grupo ha realizado mínimo un html para la página web donde 
             hemos realizado la estructura de las páginas.
* **JavaScript** - Tenemos un Script donde encontramos la información (en formato tarjeta) de los 
                   jugadores y un carrusel que se mueve segun al botón y el punto que clickes, así
                   cuando elijas la útima imagen volverá a la primera respectivamente.

## 👁️ Abrir y visualizar

### En las páginas
Dentro de las páginas, en el **header**, se encuentran los nombres de las páginas, los cuales sirven
para desplazar se entre ellas y el footer con links asociados a cada título dado.
