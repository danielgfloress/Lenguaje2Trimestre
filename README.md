## 🖥️ Descripción del proyecto

Nuestra página web sigue los partidos de futbol de los equipos del momento (Ráimon, Royal Academy, 
Academia Zeus, Wild, Farm, ...). Mostramos sus estadisticas, su posicionamiento en el campo a la 
hora de jugar, entre muchos datos de los equipos e individualmente.
También tenemos la tienda oficial de recuerdos y productos de los equipos y partidos que han jugado.

## ⚙️ Estructura y descripción de uso

* **css** - Un css común para todo el proyecto, usado por todos los integrantes del grupo,
            donde hemos combinado estilos para desarrollar una página en común.
* **html** - Cada miembro del grupo ha realizado mínimo un html para la página web donde 
            hemos realizado la estructura de las páginas.
* **js** - Tenemos un js donde encontramos la información (en formato tarjeta) de los 
            jugadores y un carrusel que se mueve segun al botón que clickes.

## 👁️ Abrir y visualizar
